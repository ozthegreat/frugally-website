# Where can I view my usage, bills, and savings?

frugally.app provides several places to view your current and historical usage, spending, and savings data.

---

## Savings and execution data

| What you want to see | Where to find it |
|---------------------|-----------------|
| **Real-time savings summary** | [Dashboard](https://dashboard.frugally.app) — the main dashboard shows total savings, active schedules, and recent execution results |
| **Projected vs actual savings** | [Dashboard](/docs/dashboard/projected-vs-actual-savings) — compare what your schedules are expected to save against actual measured savings |
| **Execution history** | [Executions page](https://dashboard.frugally.app/executions) — detailed log of every execution with per-resource results |
| **Insights and analytics** | [Insights page](https://dashboard.frugally.app/insights) — trends, savings over time, and execution analytics. See [Insights](/docs/automate/insights) |

---

## AWS cost data

| What you want to see | Where to find it |
|---------------------|-----------------|
| **AWS spending breakdown** | [Cost Explorer](https://dashboard.frugally.app/cost-explorer) — analyse spending by service, account, tag, or time period. See [Cost Explorer](/docs/intelligence/cost-explorer) |
| **RI/SP commitment usage** | [Commitment Utilisation](https://dashboard.frugally.app/commitment-utilisation) — track Reserved Instance and Savings Plan utilisation. See [Commitment Utilisation](/docs/intelligence/commitment-utilisation) |
| **Cost anomalies** | [Cost Anomaly Detection](/docs/intelligence/cost-anomaly-detection) — automated alerts for unexpected spending changes |
| **Budget tracking** | [Guard Budgets](https://dashboard.frugally.app/guard/budgets) — set spending limits and track actual spend against them. See [Budgets](/docs/guard/budgets) |

---

## Billing and credits

| What you want to see | Where to find it |
|---------------------|-----------------|
| **Credit balance** | **Settings** > **Billing** — current credit balance and plan details |
| **Credit usage history** | **Settings** > **Billing** > **Usage** — see what consumed your credits. See [Credit Usage and Enforcement](/docs/billing/credit-usage-and-enforcement) |
| **Invoices** | **Settings** > **Billing** > **Invoices** — download current and past invoices. See [Payment Methods and Invoices](/docs/billing/payment-methods-and-invoices) |
| **Plan details** | **Settings** > **Billing** > **Plan** — current plan, limits, and upgrade options. See [Plans and Credits](/docs/billing/plans-and-credits) |

---

## Exporting data

You can export data from several pages:

- **Cost Explorer** — Export cost breakdowns as CSV
- **Executions** — Export execution history as CSV
- **Insights** — Export savings reports as CSV or PDF

For programmatic access, see the [API Reference](/docs/api-reference/overview).
