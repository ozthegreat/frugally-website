# Team Management Overview

Team Management covers how you add and remove team members, configure security settings, and manage feature flags for your frugally.app workspace.

[Manage Team](https://dashboard.frugally.app/settings/team)

---

## What you can manage

| Area | Description |
|------|-------------|
| **[Members & Invitations](./members-and-invitations.md)** | Invite new members, manage pending invitations, and remove users |
| **[Security Settings](./security-settings.md)** | Session timeouts, enforced MFA, IP allowlisting, and SSO enforcement |
| **[Feature Flags](./feature-flags.md)** | Enable or disable optional features for your team |

---

## Who can manage teams

| Action | Required role |
|--------|--------------|
| View team members | All roles |
| Invite / remove members | Admin or Owner |
| Change member roles | Admin or Owner |
| Configure security settings | Admin or Owner |
| Manage feature flags | Admin or Owner |
| Transfer ownership | Owner only |

For full role details, see [Roles and Permissions](/docs/roles-and-access/roles-and-permissions).
